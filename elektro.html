Jurusan Teknik Elektro

Program Studi D3 Teknik Listrik :

Memiliki konsentrasi Instalasi Listrik dan Elektronika, bertujuan menghasilkan lulusan dengan kompetensi; mampu merancang, melaksanakan pemasangan, memelihara/memperbaiki dan pengamanan instalasi penerangan maupun instalasi tenaga, mampu melakukan pengawasan proyek instalasi listrik tegangan rendah maupun menengah, mampu melaksanakan perawatan dan perbaikan alat-alat listrik rumah tangga, instalasi alat-alat komunikasi dan telekomunikasi, peralatan elektronik, dan kontrol elektronik.

Program Studi D3 Manajemen Informatika :

Menyiapkan lulusan agar mempunyai kompetensi; mampu merakit, mengoperasikan, dan memelihara perangkat komputer, mampu merancang dan membangun program atau sistem informasi berupa program aplikasi maupun web (single user maupun multiuser), mampu menguasai teknik dan bahasa pemrograman untuk berbagai kebutuhan bisnis dan industri seperti sistem informasi manajemen sumber daya manusia, perbankan, pemasaran, penggajian, penjualan, dan sistem informasi akademis.
